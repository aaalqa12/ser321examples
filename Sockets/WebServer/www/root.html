You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html

API Endpoints

  • /multiply?num1=X&num2=Y - Multiplies two numbers
  • /github?query=users/username/repos - Lists GitHub repositories
  • /uppercase?text=STRING×=N - Converts text to uppercase N times
  • /factorial?number=X - Calculates factorial (0 ≤ X ≤ 20)
